คู่มือการใช้งาน & สรุป Logic

ปลั๊กอิน Navisworks Clash Reporter สู่ Excel

1. สรุป Logic การทำงานของโปรแกรม (Core Logic)

1 การดึงชื่อไปกรอก (View Extraction)

  • โปรแกรมจะสแกนหา Saved Viewpoints ทั้งหมด
  • ข้าม (Ignore): View ที่ถูกบันทึกไว้ลอยๆ ที่หน้า Root (ไม่ได้อยู่ในโฟลเดอร์ใดๆ) จะถูกข้ามทั้งหมด
  • ชื่อ View จะถูกทำความสะอาดช่องว่างและตัวอักษรพิเศษที่ผิดปกติ เพื่อให้การค้นหาใน Excel แม่นยำ

2 การดึงรูปภาพไปวาง (Image Processing)

  • โหมด 1 (API): โปรแกรมจะหมุนมุมกล้องไปยัง View นั้นๆ โดยอัตโนมัติ สั่งให้ Navisworks บันทึกภาพหน้าจอความละเอียดสูง (400x300) ลงในโฟลเดอร์ชั่วคราว (Temp) เพื่อนำมาใส่ Excel จากนั้นจะลบโฟลเดอร์ทิ้ง
  • โหมด 2 (XML/HTML): โปรแกรมจะเข้าไปอ่านโค้ดในไฟล์ HTML ที่คุณ Export มา เพื่อหาพาร์ทรูปภาพ () โดยจับคู่ชื่อให้ตรงกับข้อมูลใน XML อย่างแม่นยำ (รองรับภาษาไทย)
  • เมื่อนำรูปมาใส่ใน Excel โปรแกรมจะยืดขยายขนาดแถว (Row Height) ให้พอดีกับรูปภาพเป๊ะๆ

3 การจัดหมวดหมู่โฟลเดอร์ (Visual / Clash / ETC)

  • พิจารณาจาก ตัวอักษรตัวแรก ของชื่อ View เท่านั้น
  • C... = "CLASH CHECK"
    V... = "VISUAL CHECK"
    ตัวอื่นๆ = "ETC"

4 การตรวจสอบวันที่ (Date Extraction)

  • โปรแกรมจะไล่เช็ค ชื่อโฟลเดอร์แม่ จากจุดที่ View อยู่ ไล่ขึ้นไปจนถึง Root
  • มองหาชื่อโฟลเดอร์ที่เป็น ตัวเลข 8 หลักรวด (เช่น 20230526)
  • เมื่อเจอ จะแปลง Format เป็น DD/MM/YYYY ลงในรายงาน

5 การตรวจสอบ Folder "RESOLVED"

  • เช็คชื่อโฟลเดอร์แม่ทุกลำดับชั้น (ตั้งแต่ Root ลงมา)
  • หากมีโฟลเดอร์ใดก็ตามที่ชื่อเขียนว่า RESOLVED (ตัวเล็กหรือใหญ่ก็ได้)
  • รายการ View ภายใต้โฟลเดอร์นั้นทั้งหมด จะถูกทำเครื่องหมายเป็น "แก้ไขแล้ว" (IsResolved = true) ทันที

6 การอัปเดตสถานะ (Status Update)

  • เทียบรายการใหม่ กับรายชื่อที่มีอยู่แล้วในคอลัมน์ C ของ Excel
  • ถ้าเป็นรายการใหม่ทั้งหมด จะนำมาต่อท้ายให้ (Append)
  • ข้ามสถานะ "Pending": ถ้ารายการเดิมใน Excel มีสถานะปัจจุบัน (คอลัมน์ H) เป็นคำว่า "Pending" โปรแกรมจะ ไม่เขียนทับสถานะ และปล่อยไว้ตามเดิมเพื่อให้คุณไปตรวจสอบเอง
  • ตรรกะการประเมินสถานะ:
    1. หากอยู่ในโฟลเดอร์ RESOLVED สถานะ = "RESOLVED"
    2. หากไม่อยู่ใน RESOLVED ให้เช็ควันที่:
      • ถ้าวันที่ (Date) = วันที่ล่าสุด (Max Date) ของรอบนั้น -> "NEW"
      • ถ้าเป็นวันที่เก่า หรือไม่มีวันที่ -> "ACTIVE"

7 การเลือกระบุอาคาร (Building Selector)

  • โปรแกรมจะดึงชื่อ โฟลเดอร์ชั้นนอกสุด (Root Folder) มาแสดงเป็นรายชื่ออาคารในหน้า UI ให้คุณเลือกก่อนรัน
  • เมื่อคุณเลือกอาคารใดอาคารหนึ่ง โปรแกรมจะเจาะลึกเข้าไปอ่านข้อมูลและถ่ายรูป เฉพาะ View ที่อยู่ภายใต้อาคารนั้นเท่านั้น โดยข้ามข้อมูลของตึกอื่นๆ ออกไปทั้งหมด

3. คู่มือการใช้งานสำหรับ User (User Manual)

🚀

โหมด 1: แบบดึงข้อมูลโดยตรง (API Mode)

โหมดนี้สะดวกที่สุด เพราะไม่ต้อง Export ไฟล์ใดๆ ออกมาก่อน ให้ปลั๊กอินจัดการให้หมด

1
เปิดโมเดลงานในโปรแกรม Navisworks
2
เปิดโปรแกรม ปลั๊กอิน MTC Clash Reporter (จากแท็บ Ribbon)
3
เลือก ไฟล์ Excel ต้นฉบับ และ Sheet ที่ต้องการนำข้อมูลไปใส่
4
ที่หัวข้อ 3 ให้เลือก ชื่ออาคาร (Building) ที่ต้องการดึงข้อมูล
5
กดปุ่ม เริ่มอัปเดตข้อมูลลง Excel
หมายเหตุระหว่างการรัน: หน้าจอโมเดลจะมีการกะพริบและหมุนกล้องไปมาอัตโนมัติ (เพื่อถ่ายรูป) กรุณารอสักครู่จนกว่าจะมีข้อความแจ้งเตือนว่าเสร็จสมบูรณ์
📁

โหมด 2: แบบใช้ไฟล์ส่งออก (XML/HTML Mode)

โหมดนี้เหมาะสำหรับกรณีที่คุณมีการเซฟ Report จาก Navisworks เก็บไว้ล่วงหน้าแล้ว หรือต้องการทำงานออฟไลน์

1
ทำการสั่ง Export Clash Report ออกมาจาก Navisworks ให้อยู่ในรูปแบบ XML
2
ทำการสั่ง Export Clash Report ออกมาจาก Navisworks ให้อยู่ในรูปแบบ HTML (Tabular)
3
เปิดโปรแกรม ปลั๊กอิน MTC Clash Reporter และเลือกโหมด "ใช้ไฟล์ XML/HTML"
4
กดปุ่ม Browse เพื่อเลือก:
  • ไฟล์ XML
  • โฟลเดอร์ที่มีไฟล์ HTML (เลือกโฟลเดอร์ที่เก็บไฟล์ HTML เอาไว้)
  • ไฟล์ Excel ที่ต้องการอัปเดต
5
กด โหลดข้อมูลตึก เพื่อให้โปรแกรมอ่านรายชื่อตึกจากไฟล์ XML แล้วเลือก อาคาร ที่ต้องการ
6
กดปุ่ม เริ่มอัปเดตข้อมูลลง Excel และรอโปรแกรมประมวลผลจนเสร็จ

4. การติดตั้งเพื่อแชร์ให้ทีมงาน (Deployment)

ไฟล์ที่จำเป็นต้องใช้

คุณจะต้องก๊อปปี้ไฟล์ 2 ไฟล์นี้ส่งให้ทีมงาน:

  • NavisworksClashReporter.dll (ตัวปลั๊กอินหลัก)
  • HtmlAgilityPack.dll (ตัวช่วยอ่านภาษาไทย)

วิธีติดตั้งลงเครื่อง (ไม่ต้องใช้สิทธิ์ Admin)

1
ให้ทีมงานสร้างโฟลเดอร์ชื่อ NavisworksClashReporter ไว้ที่ปลายทาง
2
นำไฟล์ DLL ทั้ง 2 ไฟล์ไปใส่ไว้ในโฟลเดอร์ที่สร้างขึ้น
3
นำโฟลเดอร์ไปวางไว้ที่:
C:\User\Username\%APPDATA%\Roaming\Autodesk Navisworks Manage 2023\Plugins\(หากไม่มี Folder : Plugins ให้สร้างขึ้นมาใหม่)
💡 ทริคสำหรับการเข้าถึงโฟลเดอร์:
พิมพ์ %APPDATA% ในช่องค้นหาที่อยู่ของ File Explorer (ช่อง Address Bar) แล้วกด Enter ระบบจะพาไปที่โฟลเดอร์ AppData\Roaming อัตโนมัติ จากนั้นให้หาโฟลเดอร์ Autodesk และเข้าไปวางปลั๊กอินได้เลยครับ
