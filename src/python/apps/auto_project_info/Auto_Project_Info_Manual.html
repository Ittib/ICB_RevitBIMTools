สคริปต์ Auto Project Info (Revit Python / Dynamo)

1. สรุปฟีเจอร์เด่นและการทำงาน (Core Logic)

สคริปต์ auto_project_info.py ทำหน้าที่อ่านข้อมูลจากไฟล์ Excel แล้วนำมากรอกลงในช่อง Project Information ของ Revit โดยอัตโนมัติ ซึ่งมีจุดเด่นคือ:

1 การเชื่อมต่อและอ่านข้อมูล Excel (Excel Interop)

  • โปรแกรมจะเรียกหน้าต่างขึ้นมาให้ผู้ใช้ เลือกไฟล์ Excel (.xlsx) ที่เก็บข้อมูลผู้ออกแบบเอาไว้
  • ทำงานอยู่เบื้องหลัง (Background Process) ทำให้หน้าต่าง Excel ไม่เด้งรบกวนการทำงานของผู้ใช้
  • เจาะจงอ่านข้อมูลจาก Sheet ที่ชื่อว่า "Eng name" เท่านั้น โดยเริ่มอ่านตั้งแต่ แถวที่ 4 ไปจนถึงแถวที่ 100

2 ตรรกะการจับคู่ข้อมูล (Parameter Mapping)

  • ชื่อตำแหน่ง (Parameter Name): อ่านจากคอลัมน์ A (เช่น "Architect 1")
  • ชื่อ-นามสกุล (Thai Name): อ่านจากคอลัมน์ D แล้วนำไปกรอกลงในพารามิเตอร์ชื่อ {ชื่อตำแหน่ง} ใน Revit
  • เลขที่ใบอนุญาต (License No.): อ่านจากคอลัมน์ C แล้วนำไปกรอกลงในพารามิเตอร์ชื่อ {ชื่อตำแหน่ง} License No. ใน Revit (เช่น "Architect 1 License No.")
  • หากบรรทัดไหนไม่มีชื่อตำแหน่งในคอลัมน์ A โปรแกรมจะข้ามแถวนั้นไปทันที

3 ระบบ Transaction อัตโนมัติ

  • ใช้ TransactionManager เพื่อรวมการกรอกข้อมูลนับร้อยรายการไว้ในคำสั่งเดียว ทำให้กด Undo (Ctrl+Z) ครั้งเดียวเพื่อย้อนกลับได้หากกรอกข้อมูลผิดพลาด
  • มีระบบป้องกันไฟล์ค้าง (Cleanup) ปิดการทำงานของแอป Excel ที่ทำงานอยู่เบื้องหลังเสมอไม่ว่าจะทำงานสำเร็จหรือเกิดข้อผิดพลาด

2. วิธีการใช้งานสำหรับ User (User Workflow)

เนื่องจากปัจจุบันยังเป็น Python Script ที่รันผ่าน Dynamo (หรือเครื่องมือรันสคริปต์) การใช้งานจึงมีลำดับดังนี้:

1
เปิดโมเดล Revit ที่ต้องการกรอกข้อมูล Project Info
2
รันสคริปต์ Auto Project Info ผ่านทาง Dynamo Player (หรือหน้าต่างรันสคริปต์)
3
หน้าต่าง "กรุณาเลือกไฟล์ Excel สำหรับข้อมูลผู้ออกแบบ" จะเด้งขึ้นมา
4
เลือกไฟล์ Excel ต้นฉบับที่มี Sheet ชื่อ Eng name แล้วกด Open
5
รอสคริปต์ทำงานสักครู่ (ทำงานแบบเงียบๆ เบื้องหลัง)
6
ไปที่ Manage > Project Information ใน Revit เพื่อตรวจสอบรายชื่อและเลขใบประกอบวิชาชีพที่ถูกกรอกไว้เรียบร้อยแล้ว
✅ ข้อแนะนำ:
เพื่อให้โปรแกรมทำงานได้สมบูรณ์ ใน Project Information ของ Revit ควรมีการสร้าง Shared Parameter หรือ Project Parameter ที่ชื่อตรงกับคอลัมน์ A ไว้ล่วงหน้าแล้ว ทั้งตัวชื่อคน (เช่น Architect 1) และเลขใบอนุญาต (เช่น Architect 1 License No.)

การแปลงร่างเป็น Revit Add-in (.dll)

ในอนาคต เครื่องมือนี้จะถูกอัปเกรดจาก Python Script ธรรมดาให้กลายเป็นปลั๊กอิน (C# DLL) เต็มรูปแบบ ซึ่งจะส่งผลดีดังนี้:

  • ใช้งานง่ายขึ้นเพียงคลิกเดียวจากแถบเมนู Add-ins (ไม่ต้องยุ่งกับ Dynamo อีกต่อไป)
  • หน้าต่าง UI จะสวยงามและใช้งานง่ายกว่าเดิม
  • ทำงานเร็วขึ้น และไม่ต้องพึ่งพา Excel Interop แบบเดิม (อาจเปลี่ยนไปใช้ไลบรารีอ่านไฟล์ Excel โดยตรงที่รวดเร็วกว่า)
