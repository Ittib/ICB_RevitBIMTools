# Revit Family Parameter Extractor

ชุดสคริปต์สำหรับดึงข้อมูลพารามิเตอร์ของ Revit Family และ Type ต่างๆ ส่งออกไปยังไฟล์ Excel อย่างเป็นระเบียบโดยอัตโนมัติ ออกแบบมาเพื่อรันผ่าน Python Script Node ภายใน Dynamo ของ Revit

## ความสามารถหลัก (Key Features)
- ดึงข้อมูลพารามิเตอร์ได้ลึกถึงระดับ Type และแสดงค่าได้อย่างถูกต้อง
- ดึงข้อมูลได้ทั้งแบบ Family ที่อยู่ในโปรเจกต์ (Active Project) และไฟล์ .rfa ที่อยู่ข้างนอกโปรเจกต์
- จัดกลุ่มพารามิเตอร์ให้อัตโนมัติใน Excel พร้อมทำ Group (Collapse/Expand)
- **ระบบแยกสีอัจฉริยะ (Color Coding) ใน Excel:**
  - 🟦 **สีน้ำเงิน (Dark Blue):** BuiltIn / System Parameter (พารามิเตอร์พื้นฐานของระบบ Revit)
  - 🟩 **สีเขียว (Dark Green):** Family Parameter (พารามิเตอร์ที่สร้างขึ้นเฉพาะภายใน Family นั้น)
  - 🟧 **สีส้ม (Dark Orange):** Shared Parameter (พารามิเตอร์ที่ดึงมาจาก Shared Parameter File)
  - ⬛ **สีเทา/สีดำ:** ข้อมูลทั่วไป (Path ไฟล์, ชื่อ Family, ชื่อ Type)

## รายละเอียดของสคริปต์แต่ละไฟล์

ไฟล์ถูกแบ่งออกเป็น 5 ไฟล์ตามรูปแบบการใช้งาน เพื่อให้สามารถเลือกใช้ได้สะดวกที่สุด

### 🌟 00_Combined_AllModes.py (แนะนำให้ใช้)
เป็นไฟล์แบบ **All-in-One** ที่รวมความสามารถของทุกโหมดเข้าไว้ด้วยกัน เมื่อรันผ่าน Dynamo จะมีหน้าต่าง UI หลักปรากฏขึ้นมาให้คุณเลือกว่าต้องการดึงข้อมูลด้วยวิธีการใด (ใช้งานง่ายที่สุด เหมาะกับการรันผ่าน Dynamo Player)

### 📂 01_ActiveProject_ByFamily.py (โหมดเลือกเจาะจง Family)
- **การทำงาน:** ดึงข้อมูลจากโปรเจกต์ที่กำลังเปิดอยู่ (Active Project)
- **รูปแบบ:** จะมีหน้าต่าง UI ขึ้นมาให้คุณเลือกติ๊ก **"รายชื่อ Family"**
- **ผลลัพธ์:** Excel จะสร้าง Sheet แยก 1 Sheet ต่อ 1 Family และแสดงพารามิเตอร์ของ **"ทุก Type"** ที่อยู่ภายใน Family นั้น เรียงลำดับชื่อ Type ตามตัวอักษร (A-Z)

### 📁 02_ActiveProject_ByCategory.py (โหมดเหมารวมตามหมวดหมู่)
- **การทำงาน:** ดึงข้อมูลจากโปรเจกต์ที่กำลังเปิดอยู่ (Active Project)
- **รูปแบบ:** จะมีหน้าต่าง UI ขึ้นมาให้คุณเลือกติ๊ก **"Category"** (เช่น Doors, Windows, Furniture)
- **ผลลัพธ์:** Excel จะสร้าง Sheet แยก 1 Sheet ต่อ 1 Category และดึงพารามิเตอร์ของ **"Type ปัจจุบัน (Current Type)"** ของทุก Family ที่สังกัดอยู่ใน Category นั้นๆ

### 📚 03_ExternalFolder_Batch.py (โหมดประมวลผลเหมาทั้งโฟลเดอร์)
- **การทำงาน:** ดึงข้อมูลจากไฟล์ `.rfa` ที่บันทึกอยู่ในเครื่องโดย **ไม่ต้องโหลดเข้าโปรเจกต์**
- **รูปแบบ:** จะมีหน้าต่างให้เลือก **"โฟลเดอร์หลัก"** (รองรับการสแกนทะลุซับโฟลเดอร์)
- **ผลลัพธ์:** Excel จะรวบรวมข้อมูลของทุก Family มาไว้ใน Sheet เดียว โดยดึงข้อมูลของ **"Type ปัจจุบัน (Current Type)"** และจะมีการจัดกลุ่มแถว (Row Grouping) ตามลำดับชั้นของซับโฟลเดอร์ให้ด้วย ทำให้ทราบว่าไฟล์มาจากโฟลเดอร์ไหน

### 📄 04_ExternalFile_Multiple.py (โหมดดึงข้อมูลเฉพาะไฟล์ที่เลือก)
- **การทำงาน:** ดึงข้อมูลจากไฟล์ `.rfa` ที่บันทึกอยู่ในเครื่อง โดยสามารถเลือกได้ 1 ไฟล์ หรือ "หลายๆ ไฟล์" พร้อมกัน
- **รูปแบบ:** จะมีหน้าต่างให้เลือกไฟล์ `.rfa` (คุณสามารถใช้เมาส์ลากคลุมหรือกด Ctrl/Shift เพื่อเลือกหลายไฟล์ได้)
- **ผลลัพธ์:** Excel จะสร้าง 1 Sheet สำหรับครอบคลุมทุก Family ที่เลือกมา และดึงพารามิเตอร์ของ **"ทุก Type"** ในไฟล์เหล่านั้นออกมาแจกแจงอย่างละเอียด พร้อมเรียงลำดับชื่อ Type ตามตัวอักษร (A-Z)

## 🛠 วิธีการนำไปใช้งานร่วมกับ Dynamo

1. เปิดโปรแกรม Revit และสร้างโปรเจกต์ใหม่ หรือเปิดโปรเจกต์ที่มีเนื้อหา/Family ที่คุณต้องการดึงข้อมูล
2. ไปที่แท็บ **Manage** > กดปุ่มเปิด **Dynamo**
3. ค้นหาและวาง Node ที่ชื่อว่า `Python Script` ลงในพื้นที่ทำงาน (Workspace)
4. วาง Node ที่ชื่อว่า `Boolean` นำไปต่อเข้าที่ช่อง `IN[0]` ของ Python Script (และตั้งค่าสวิตช์เป็น `True`)
5. ดับเบิ้ลคลิกที่ Node Python Script จะมีหน้าต่างเขียนโค้ดเด้งขึ้นมา ให้ลบโค้ดเริ่มต้นทิ้งให้หมด
6. นำโค้ดจากไฟล์ เช่น `00_Combined_AllModes.py` คัดลอกไปวาง (Paste) ลงในหน้าต่างเขียนโค้ด แล้วกด **Save Changes**
7. กดปุ่ม **Run** ที่มุมซ้ายล่างของหน้าจอ Dynamo
8. รอสักครู่ จะมีหน้าต่าง (UI) ปรากฏขึ้นมากลางหน้าจอ ให้ท่านเลือกโหมด/เลือกข้อมูลตามต้องการ
9. เมื่อกดตกลง โปรแกรมจะรันกระบวนการดึงข้อมูลและเปิดโปรแกรม Excel ขึ้นมาแสดงผลให้ท่านทันที!

## ข้อควรระวัง (Troubleshooting)
- หากข้อมูลมีจำนวนมาก การประมวลผล Excel อาจใช้เวลาสักครู่ โปรแกรมจะทำการหยุดการกระพริบหน้าจอ Excel ไว้ชั่วคราวเพื่อให้เขียนข้อมูลได้เร็วที่สุด **กรุณารอจนกว่าจะมี Pop-up แจ้งเตือนว่าทำงานเสร็จสมบูรณ์**
- หากพบปัญหาหน้าต่างโฟลเดอร์หรือ UI ไม่ยอมขึ้นมา ให้ลองเช็คว่าเปิดโปรแกรมซ้อนทับหรือมีหน้าต่างไปซ่อนอยู่ด้านหลังโปรแกรม Revit หรือไม่
