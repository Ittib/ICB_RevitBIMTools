# 📊 Revit Family Parameter Extractor
**เครื่องมือดึงข้อมูลและพารามิเตอร์จาก Revit Family แบบอัตโนมัติ สู่ตาราง Excel**

</div>

---

## 🌟 จุดเด่นของโปรแกรม
โปรแกรมนี้จะช่วยลดเวลาและเพิ่มความแม่นยำในการจัดการข้อมูลโมเดลของคุณ โดยสามารถดึงพารามิเตอร์ต่างๆ จากไฟล์ Revit Family ให้ออกมาจัดเรียงในตาราง Excel ได้อย่างสวยงามและเป็นระเบียบ พร้อมใช้งานทันที

---

## 🎨 ความหมายของสีในตาราง Excel (Color Codes)
ตาราง Excel ที่ส่งออกจะมีการแยกสีพื้นหลัง (Highlight) บนหัวคอลัมน์เพื่อให้คุณทราบถึงประเภทของพารามิเตอร์ได้ในพริบตา:

| สีใน Excel | ประเภทพารามิเตอร์ | ความหมาย |
| :---: | :--- | :--- |
| 🟦 **น้ำเงิน** | **BuiltIn / System** | พารามิเตอร์พื้นฐานที่ฝังมากับระบบของ Revit |
| 🟩 **เขียว** | **Family Parameter** | พารามิเตอร์ที่ถูกสร้างขึ้นเฉพาะใน Family นั้นๆ (Local) |
| 🟧 **ส้ม** | **Shared Parameter** | พารามิเตอร์ที่ดึงมาจากไฟล์ Shared Parameter (.txt) |
| ⬛ **เทา/ดำ** | **General Info** | ข้อมูลทั่วไปที่ระบบสร้างขึ้น เช่น ชื่อ Family, ชื่อ Type, Path |

> 💡 **Tip:** การแยกสีช่วยให้คุณทำ Data Audit ว่า Family ของคุณได้ใช้ Shared Parameter อย่างถูกต้องหรือไม่ได้อย่างรวดเร็ว

---

## 🚀 โหมดการทำงานหลัก (4 Modes)

### 1️⃣ Active Project: Select by Family
*ดึงข้อมูลลึกระดับ Type จากโปรเจกต์ที่เปิดอยู่*
- **เหมาะสำหรับ:** ตรวจสอบข้อมูล Family ที่โหลดเข้ามาในโปรเจกต์ปัจจุบัน
- **การทำงาน:** จะมีหน้าต่างขึ้นมาให้คุณคลิกเลือกรายชื่อ Family ที่ต้องการได้ทีละหลายๆ ตัว
- **ผลลัพธ์:** สร้างไฟล์ Excel **แยก 1 Sheet ต่อ 1 Family** และดึงข้อมูลของ **"ทุก Type"** ออกมาแจกแจงอย่างละเอียด

### 2️⃣ Active Project: Select by Category
*ภาพรวมของแต่ละหมวดหมู่*
- **เหมาะสำหรับ:** การทำ BOQ, Schedule แบบรวดเร็ว หรือเช็คพารามิเตอร์แบบเหมารวมทั้ง Category (เช่น ประตู, หน้าต่าง ทั้งโปรเจกต์)
- **การทำงาน:** เลือกหมวดหมู่ (Category) ที่ต้องการ
- **ผลลัพธ์:** สร้างไฟล์ Excel **แยก 1 Sheet ต่อ 1 หมวดหมู่** และดึงเฉพาะ **"Type ที่กำลังถูกใช้งานอยู่"** ในโมเดลมาแสดง

### 3️⃣ External Files: Batch process Folder (.rfa)
*จัดการคลัง Family ขนาดยักษ์*
- **เหมาะสำหรับ:** ผู้ที่ต้องการจัดการ Library หรือมีไฟล์ Family (.rfa) ในคอมพิวเตอร์เป็นจำนวนมาก
- **การทำงาน:** เลือก "โฟลเดอร์" ที่เก็บไฟล์ไว้ (รองรับการค้นหาในซับโฟลเดอร์ย่อยทั้งหมด)
- **ผลลัพธ์:** ข้อมูลของทุกไฟล์จะถูกสรุปรวมใน **Excel หน้าเดียว** (ดึงเฉพาะ **"Type ปัจจุบัน/ค่าเริ่มต้น"**) พร้อมการจัดกลุ่มและระบุที่มาของไฟล์ (File Path) เพื่อความสะดวกในการจัดการ

### 4️⃣ External Files: Select Multiple Files (.rfa)
*ทำงานเฉพาะจุด ยืดหยุ่นสูงสุด*
- **เหมาะสำหรับ:** การเจาะจงดึงข้อมูลแบบละเอียดจากไฟล์ .rfa ที่ต้องการเพียงแค่บางไฟล์ (ไม่ต้องเปิดโปรเจกต์)
- **การทำงาน:** เลือกไฟล์ .rfa ตามต้องการ (กด Ctrl/Shift เพื่อเลือกหลายไฟล์)
- **ผลลัพธ์:** 
  - ข้อมูล **"ทุก Type"** ของไฟล์ที่เลือกจะถูกดึงออกมา
  - 🌟 **อัปเดตใหม่:** คุณสามารถเลือกได้ว่าจะ **"รวมทุก Family ไว้ในหน้าเดียว"** หรือ **"แยก Sheet ตามชื่อ Family"** (พร้อมเปลี่ยนสีแท็บ) ผ่านตัวเลือกในโปรแกรม
