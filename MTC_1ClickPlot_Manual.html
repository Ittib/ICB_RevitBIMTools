คู่มือการใช้งาน & สรุป Logic

ปลั๊กอิน MTC 1ClickPlot Pro (Revit C# Add-in)

1. สรุปฟีเจอร์เด่นและการทำงาน (Core Logic)

โปรเจกต์นี้เป็น Add-in สำหรับ Autodesk Revit ทำหน้าที่จัดการการพิมพ์แบบ (Plot) ออกเป็น PDF แบบอัตโนมัติ โดยมีจุดเด่นคือ:

1 การแปลงสีชั่วคราว (Temporary Black & White)

  • โปรแกรมจะเข้าถึง Object Styles ของโปรเจกต์ และ เปลี่ยนสีเส้นทั้งหมดให้กลายเป็นสีดำชั่วคราว เมื่อสั่ง Print
  • เมื่อทำการพิมพ์เสร็จสิ้น โปรแกรมจะ คืนค่าสีเดิม (Revert) ให้โมเดลกลับสู่สภาพปกติอัตโนมัติ

2 ตรวจจับคิวพิมพ์อัตโนมัติ (Spooler Detection)

  • โปรแกรมสามารถ ตรวจจับคิวพิมพ์จาก Windows Spooler เพื่อตรวจสอบว่าไฟล์ถูกส่งไปที่เครื่องพิมพ์จำลอง (PDF Printer) จนเสร็จสมบูรณ์หรือยัง เพื่อป้องกันปัญหาสั่งพิมพ์ซ้อนหรือไฟล์ไม่สมบูรณ์

3 ระบบ Auto-Rename & Move

  • ตรรกะการตั้งชื่อ (Rename Logic): โปรแกรมจะดึงค่า Sheet Number (หมายเลขแผ่นแบบ) ของแต่ละ Sheet ใน Revit มาใช้เป็นชื่อไฟล์ โดยจะตัดตัวอักษรพิเศษที่ไม่สามารถตั้งชื่อไฟล์ได้ออก \ / * ? : " < > | อัตโนมัติ
  • การทำงาน: โปรแกรมจะค้นหาไฟล์ใน Temp Folder ที่มีคำตรงกับ Sheet Number แล้วทำการเปลี่ยนชื่อเป็น {Sheet Number}.pdf จากนั้นจะ ย้ายไฟล์ไปยังโฟลเดอร์ปลายทาง (Output Directory) ที่ผู้ใช้งานเลือกไว้ให้แบบอัตโนมัติ

2. วิธีการใช้งานสำหรับ User (User Workflow)

ขั้นตอนตั้งแต่เปิดหน้าต่าง UI จนไปถึงได้ไฟล์ PDF ออกมา มีลำดับดังนี้:

1
เปิด Revit เปิดโปรเจกต์ และคลิกปุ่ม MTC 1ClickPlot บนแถบเมนู Add-in
2
เลือกหมวดหมู่ที่ต้องการยกเว้น: ติ๊กเลือก Object Styles (เช่น Revision Clouds) ที่ ไม่ต้องการ ให้เปลี่ยนเป็นสีดำ
3
เลือกเครื่องพิมพ์และตั้งค่า: เลือก Printer เป็น PDF24 และเลือก Print Settings ตามต้องการ
4
เลือก View / Sheet:
  • เลือกจาก View/Sheet Set ที่เซฟไว้ในโปรเจกต์ หรือ...
  • เลือก Custom Select เพื่อติ๊กเลือก Sheet ที่ต้องการพิมพ์ด้วยตัวเองแบบรายแผ่น
5
ตั้งค่า Output:
  • ติ๊กเลือก "รวมเป็นไฟล์เดียว (Combine)" หากต้องการ PDF ไฟล์เดียว
  • ติ๊กเลือก "เปลี่ยนชื่อไฟล์อัตโนมัติ (Auto-Rename)" หากต้องการแยกแผ่นและเปลี่ยนชื่อตาม Sheet Number
6
เลือกโฟลเดอร์ปลายทาง: กด Browse เพื่อเลือกโฟลเดอร์ Output ว่าอยากให้ไฟล์ไปเซฟที่ไหน
7
กดปุ่ม RUN PROCESS (โปรแกรมจะเปลี่ยนสี -> สั่ง Print -> รอไฟล์ -> เปลี่ยนชื่อ/ย้ายไฟล์ -> และคืนค่าสีเดิม)
📂 ผลลัพธ์ไฟล์ (Output Location):
- หากรวมไฟล์ (Combine): จะได้ไฟล์ Combined_Print.pdf ไปโผล่ที่โฟลเดอร์ปลายทางที่คุณเลือกทันที
- หากแยกไฟล์ (Auto-Rename): ไฟล์ทั้งหมดจะไปโผล่ในโฟลเดอร์ปลายทางที่เลือก โดยแยกเป็นแผ่นๆ ชื่อไฟล์คือ {Sheet Number}.pdf

3. การตั้งค่า PDF24 (PDF24 Configuration)

เพื่อให้ปลั๊กอินทำงานแบบเงียบๆ ไม่ต้องมานั่งกด Save เองทุกแผ่น ให้ตั้งค่า PDF24 ดังนี้:

1
เปิดโปรแกรม PDF24 Settings (ไอคอนรูปฟันเฟือง)
2
ไปที่เมนู PDF Printer (เครื่องพิมพ์ PDF)
3
เปิดใช้งานระบบบันทึกอัตโนมัติ (Auto-Save / บันทึกไฟล์อัตโนมัติ)
4
ตั้งค่าโฟลเดอร์ปลายทาง (Directory) ให้เป็น: C:\Revit_PDF_Temp

*จำเป็นต้องตั้งค่าให้ตรงกับ Config ในโค้ด เพื่อให้ปลั๊กอินไปดักจับไฟล์และเปลี่ยนชื่อได้อย่างถูกต้อง

4. วิธีนำไฟล์ไปติดตั้ง (Deployment)

🚀

วิธีนำไปติดตั้งลงเครื่องผู้ใช้งาน

ปลั๊กอินนี้จะมีไฟล์ที่จำเป็นสำหรับการทำงานอยู่ 2 ไฟล์ด้วยกัน ให้นำไปติดตั้งตามขั้นตอนดังนี้:

1
เตรียมไฟล์ MTC_1ClickPlot.dll (ตัวโปรแกรมหลัก)
2
เตรียมไฟล์ MTC_1ClickPlot.addin (ไฟล์สำหรับบอกให้ Revit รู้จักปลั๊กอิน)
3
นำไฟล์ทั้ง 2 อันนี้ไปวางไว้ด้วยกันที่โฟลเดอร์:
C:\Users\Username\AppData\Roaming\Autodesk\Revit\Addins\2024\
4
เปิดโปรแกรม Revit 2024 ขึ้นมา โปรแกรมจะทำการโหลด Add-in ให้พร้อมใช้งานในแถบเมนูทันที
💡 เคล็ดลับการอัปเดตสำหรับ User เดิม:
หากเคยติดตั้งไปแล้วและได้รับไฟล์ปลั๊กอินเวอร์ชันใหม่มา ไม่จำเป็นต้องยุ่งกับไฟล์ .addin อีกต่อไป ให้ปิดโปรแกรม Revit แล้วนำไฟล์ .dll ตัวใหม่ไปวางทับไฟล์เดิมได้เลย
